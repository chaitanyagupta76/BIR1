# BIR1 - Boss Baby 1st Birthday Template

A modern, production-ready 1st Birthday Invitation Website Template using Next.js + TypeScript + Tailwind CSS 4.

## Features
- Boss Baby Theme (Navy & Gold)
- Interactive Scroll Timeline
- Multi-language (English / Telugu)
- Configurable via JSON
- Background Music & Countdown Timer
- Confetti Animations

## Getting Started
1. Install dependencies: \
pm install\
2. Run development server: \
pm run dev\
3. Customize content in \src/config/content.json\
